Formatação de texto (Semântico)

Negrito / Destaque

Podemos formatar um texto em negrito (semântico) através da tag <strong>, como esse texto aqui.

itálico / Ênfase

Podemos também formatar um texto em itálico (semântico) através da tag <em> como Esse texto aqui.

Texto Marcado

Usando a tag <mark> podemos marcar um texto como esse aqui.

Texto pequeno

Usando a tag <small> podemos deixar um texto pequeno como esse aqui.

Texto deletado

Podemos usar a tag <del> para mostrar que um que um texto não deve ser considerado como esse aqui.

Texto inserido

Podemos usar a tag <ins> para mostrar um texto inserido como esse aqui.

Texto sobrescrito

Podemos usar a tag <sup> para elevar um caractere por exemplo: 220+10.

Text subscrito

Podemos usar a tag <sub> para ceder um caractere por exemplo: H2O.
